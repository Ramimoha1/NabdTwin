# Edit Visualization - Quick Start Guide

## For Administrators

### How to Edit Floor Plans

#### Step 1: Access Edit Visualization
1. Log in as an administrator
2. Click **"Edit Visualization"** in the sidebar (icon: pencil/edit)

#### Step 2: Select Branch
1. Choose a branch from the dropdown
2. View all floors for that branch

#### Step 3: Create or Edit Floor
- **To create a new floor**:
  1. Click "Add Floor" button
  2. Enter floor name (e.g., "Ground Floor")
  3. Enter floor number (e.g., 0, 1, 2)
  4. Optionally add description
  5. Click "Create Floor"

- **To edit existing floor**:
  1. Click "Edit Floor Plan" button on any floor
  2. Wait for editor to load

#### Step 4: Design Your Floor Plan
The editor provides three main tabs:

**Edit Floorplan Tab**:
- Click "Draw" to add walls
- Click points to create corners
- Double-click to finish drawing
- Use "Move" to adjust corners
- Use "Delete" to remove walls

**Design Tab**:
- View your 3D floor plan
- Rotate camera with mouse drag
- Zoom with scroll wheel
- Use camera controls (bottom right)

**Add Items Tab**:
- Drag furniture onto floor plan
- Add employee models (male/female)
- Click item to adjust size
- Click employee model to assign employee

#### Step 5: Assign Employees to Models
1. Place an employee model on the floor
2. Click the employee model to select it
3. In the right panel, find "Assign Employee" section
4. Select employee from dropdown (shows name and role)
5. Employee is now linked to that model

#### Step 6: Save Your Work
1. Click "Save Floor Plan" button (top right)
2. Wait for "Floor plan saved successfully" message
3. Click "Go Back" to return to floor list

### Tips & Best Practices

- **Start with Floorplan**: Draw walls before adding furniture
- **Close Your Walls**: Make sure walls form complete rooms
- **Use Grid**: Grid helps with alignment (visible in editor)
- **Test Different Views**: Switch between 2D and 3D views
- **Save Frequently**: Click save after major changes
- **Name Clearly**: Use descriptive floor names for easy identification

### Common Issues

**Q: Employee dropdown is empty**
- Ensure the branch has employees assigned in the system

**Q: Can't draw walls**
- Make sure you're in the "Edit Floorplan" tab
- Click "Draw" button first

**Q: Item won't move**
- Check if it's locked (uncheck "Lock in place")

**Q: Changes not saving**
- Check for error messages
- Verify you have admin permissions
- Try refreshing and editing again

### Keyboard Shortcuts (in editor)
- **Delete**: Remove selected item
- **Escape**: Deselect item
- **Mouse Drag**: Rotate camera (3D view)
- **Scroll**: Zoom in/out
